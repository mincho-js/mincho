# 소개

Mincho.js는 타입 안전하고 제로 런타임 스타일링을 위해 설계된 현대적인 CSS-in-JS 라이브러리입니다.

## 왜 Mincho.js인가?

- **제로 런타임 오버헤드**: 모든 스타일이 빌드 시점에 추출됩니다
- **타입 안전한 스타일**: IDE 자동 완성이 포함된 완벽한 TypeScript 지원
- **익숙한 문법**: JavaScript 템플릿 문자열로 CSS 리터럴 사용
- **프레임워크 무관**: React 등과 함께 작동
