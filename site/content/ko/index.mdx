# Mincho.js에 오신 것을 환영합니다

Mincho.js는 React 애플리케이션을 위한 타입 안전하고 제로 런타임 스타일링을 제공하는 CSS-in-JS 라이브러리입니다.

## 특징

- **타입 안전**: TypeScript를 완벽하게 지원하며 자동 완성 제공
- **제로 런타임**: 스타일이 빌드 시점에 추출됨
- **간단한 API**: JavaScript 템플릿 리터럴로 CSS 작성

## 빠른 시작

프로젝트에서 Mincho.js 시작하기:

```bash
npm install @mincho-js/css
```

자세한 내용은 [문서](/ko/docs)를 확인하세요.
