# Welcome to Mincho.js

Mincho.js is a CSS-in-JS library that provides type-safe, zero-runtime styling for React applications.

## Features

- **Type-safe**: Full TypeScript support with autocompletion
- **Zero-runtime**: Styles are extracted at build time
- **Simple API**: Write CSS with JavaScript template literals

## Quick Start

Get started with Mincho.js in your project:

```bash
npm install @mincho-js/css
```

Check out the [Documentation](/docs) for more details.
