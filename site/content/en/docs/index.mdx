# Introduction

Mincho.js is a modern CSS-in-JS library designed for type-safe, zero-runtime styling.

## Why Mincho.js?

- **Zero runtime overhead**: All styles are extracted at build time
- **Type-safe styles**: Full TypeScript support with IDE autocompletion
- **Familiar syntax**: Use CSS literals with JavaScript template strings
- **Framework agnostic**: Works with React, and more
